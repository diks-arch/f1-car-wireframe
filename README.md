# F1 Car Wireframe

A 3D wireframe Formula 1 car visualization built using Python and Panda3D.

## Features
- 3D wireframe-rendered F1 car model
- Custom car body, front wing, rear wing, side pods, and wheels
- Ambient and directional lighting
- Smooth 360° rotating camera
- Object-oriented Python implementation

## Technologies Used
- Python
- Panda3D

## Installation

```bash
pip install panda3d

## Run

```bash
python f1_car_wireframe.py
```

## Project Overview

This project demonstrates the creation of a Formula 1 car using basic geometric primitives in Panda3D. The model is rendered in wireframe mode and viewed through an automated rotating camera, showcasing fundamental concepts of 3D graphics, modeling, lighting, and scene management.
